微软宣布将停止对 Windows 10 的官方支持，引发了广泛的争议，因为这意味着安全更新和免费技术支持的终结。

虽然停止支持是任何软件系统生命周期中的自然阶段，但微软此次面临的批评浪潮并非源于这一决定本身，而是源于其强制用户迁移到 Windows 11 的方式。

Windows 10 发布于十年前，已被广泛应用于家庭、企业和政府机构。

停止支持标志着科技行业的一个重大转折点，迫使全球数百万用户升级到 Windows 11 或放弃他们当前的设备。

微软将 Windows 11 宣传为提升安全性和性能的必要步骤，但其严格的升级要求导致数百万台计算机无法兼容。

此次升级引发了关于计划报废和大型企业环境责任的辩论，许多用户认为这是一场灾难，因为它会带来深远的环境、经济和社会影响。

停止支持的原因

Windows 10 最初以操作系统即服务 (OSaaS) 的形式推出，通过持续更新而非重大版本更新来提供支持，微软也承诺提供长期支持。

然而，在 2021 年，微软宣布将于 2025 年 10 月停止对 Windows 10 的官方支持，最终版本为 22H2。

9 月份的数据显示，尽管其继任者 Windows 11 已于 2021 年发布，但全球仍有十分之四的 Windows 用户在使用 Windows 10。

微软停止支持的主要原因包括：专注于开发支持人工智能等高级功能的 Windows 11，以及为了节省资源而减少对旧版本的支持。

微软提供扩展安全更新 (ESU) 计划，该计划仅提供关键安全更新，不包含新功能、非安全修复或针对一般问题的技术支持。

费用和持续时间因用户类型（个人消费者或企业客户）而异。

为了赢得用户忠诚度，该公司提供了首年免费更新的选项。然而，这项服务并非完全免费，因为它需要用户将设备连接到 Windows 备份服务，并通过 OneDrive 将设置同步到云端。

微软奉行将安全服务转化为一项服务的策略，将关键安全更新与其云平台关联起来。

通常情况下，备份次数会超出免费限额，迫使用户购买月度或年度存储套餐。这样一来，免费选项实际上就变成了云服务的销售。

严格的系统要求阻碍了兼容性

Windows 11 提出了严格的技术要求，最显著的是支持安全启动 (Secure Boot) 和第二代可信平台模块 (TPM)。

支持的英特尔处理器仅限于第八代处理器，而 AMD 的支持范围则包括 Ryzen 2000 系列及以上型号。

微软声称这些要求可以增强系统安全性，抵御某些攻击，例如窃取存储密钥或篡改启动级软件。

然而，人们开始质疑这些要求是否必要且值得商榷，因为该系统虽然无法使用某些高级安全功能，但仍然可以在不满足这些要求的设备上运行。

用户已经成功地绕过了这些要求，并使用官方和非官方方法在不支持的设备上运行 Windows 11，这表明即使没有这些要求，系统也能正常运行，且不会出现重大问题。

这表明，这些要求可能是一种计划性淘汰的手段，尤其是在微软最近发布的版本中放宽了一些限制的情况下。

这些要求导致数百万台设备无法升级，即便它们性能远超基本要求，能够高效处理日常任务，从而引发了关于可持续性的争论。

有人认为，这些要求的真正目的并非提升安全性，而是为了控制设备生命周期，刺激新设备市场。

升级本应是延长设备寿命、提升效率的工具，如今却沦为人为加速设备淘汰的手段。

电子垃圾灾难

据 Canalys 的一项研究显示，停止对 Windows 10 的支持预计将导致约 2.4 亿台电脑变成电子垃圾，相当于 32 万辆汽车的重量。

回收电子垃圾需要大量的能源和专门的设施才能安全处理。

电子设备如果不进行回收，通常会被填埋，其中的铅、汞和镉等有毒物质会渗入土壤。

这些化学物质会污染土壤，影响土壤质量，使其不适宜耕种。它们还会对饮用水源和水生生态系统构成威胁。此外，焚烧电子垃圾还会向大气中释放有害气体。

许多用户对这一决定表示反对，认为这是一场环境灾难，并呼吁抵制微软或转而使用 Linux 系统。

像“重启计划”（Project Restart）这样的活动呼吁禁止微软的软件升级，他们认为这项看似以安全为重点的更新实际上可能会增加全球电子垃圾的数量。

在各大科技公司都在努力减少碳足迹之际，微软的这一举措与所有关于环境可持续性的说法背道而驰，将一次软件更新变成了一场环境灾难。

这项政策的影响不仅限于环境；它还波及经济，尤其是在依赖二手设备的欠发达国家，因为购买一台新电脑可能需要花费数百甚至数千美元。

许多用户表示，他们不希望进行强制升级，尽管存在风险，他们仍然选择继续使用Windows 10。

微软持续存在的问题

Windows 11 因隐私问题而备受诟病，因为它会收集大量诊断数据并将其发送到公司服务器，但并未充分说明这些数据的性质或用途。

新的 Outlook 应用被指控在用户使用非微软邮箱账户登录后，立即将所有邮件上传到公司服务器进行人工智能扫描。

微软照片应用也包含一个编辑按钮，该按钮会在未经用户明确确认或同意的情况下，将用户查看的照片上传到云端。

而且，问题不仅仅在于操作系统许可费用；Windows 11 还会在不应出现商业广告的地方显示付费广告和营销推荐。

微软已测试在“开始”菜单中显示广告，锁屏界面上也会出现促销广告和提醒，以及试图阻止用户切换到 Microsoft Edge 以外的浏览器的弹出窗口。

微软正以强制且密集的方式整合其人工智能工具（尤其是 Copilot）和 OneDrive 云服务，Copilot 按钮醒目地显示在任务栏和搜索工具中。

这种强制整合的做法迫使用户与微软的云和人工智能生态系统互动，即使他们并不情愿。

总之，停止对 Windows 10 的支持不仅仅是一项技术事件，更是一场环境和经济灾难，暴露了科技行业计划报废政策的弊端。尽管微软提供了一些安全方面的优势，但对于许多用户来说，升级到 Windows 11 似乎既不必要也不可取，尤其是在 Linux 等替代方案可供选择的情况下。
